# BrxDarkMenu - Dark Theme Helper for BricsCAD

BrxDarkMenu is a lightweight utility designed to enhance the native dark mode experience in BricsCAD. While BricsCAD offers a built-in dark theme, several user interface elements—such as context menus, title bars, and specific dialog icons—traditionally remain light or unoptimized. This helper patches those visual gaps to deliver a completely unified, eye-strain-free dark workspace.

## Preview

![BricsCAD Dark Theme Helper](https://github.com)

## Core Features

* **Dark Context Menus:** Transforms legacy light-themed application drop-downs and right-click menus into a sleek dark palette.
* **True Dark Title Bar:** Forces the main Windows application title bar header to respect dark system configurations.
* **Optimized Dialog Icons:** Replaces mismatched or low-contrast icons within core dialog boxes for improved visibility.

## Getting Started

### Prerequisites
* **Operating System:** Windows 10 or Windows 11 (with system Dark themse).
* **Target Application:** BricsCAD (v24 or newer recommended).

### Installation

1. Download the latest release package from the **Releases** tab.
2. Use the appload command to load the matching .BRX for your BricsCAD version
